# Processed Modeling Dataset

Generated by:

```powershell
python scripts/build_modeling_dataset.py
```

Local output files:

- `data/processed/nba_player_seasons_modeling.csv`
- `data/processed/preprocessing_summary.json`

These files are ignored by Git because they are reproducible from the raw Kaggle snapshot.

## Cleaning Rules

- Source table: `data/raw/Player Totals.csv`
- League filter: `lg == "NBA"`
- Unit of observation: one player-season
- Group key: `season`, `player_id`
- Multi-team summary rows such as `2TM`, `3TM`, `4TM`, and `5TM` are excluded before manual aggregation.
- Team-specific rows are summed for counting stats.
- Per-game features are recomputed as aggregated total divided by games played.
- Shooting percentages are recomputed from aggregated makes and attempts.
- Pre-1980 three-point makes and attempts are treated as zero because the NBA three-point line did not exist yet.
